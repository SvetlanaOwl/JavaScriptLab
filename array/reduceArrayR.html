JavaScript Arrays

The reduce() method

.reduce() принимает функцию с двумя параметрами: acc — аккумулятор (накопитель результата), num — текущий элемент массива.

Вторым аргументом указывается начальное значение аккумулятора.

После прохода по массиву возвращается одно итоговое значение.

.reduce() — универсальный метод, который может заменить map, filter и даже forEach, если нужно собрать данные в одно целое.
